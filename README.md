# Uniswap Trade

## Requirements
- Node 20 or greater

## Setup
`.env`
```
MAINNET_URL=https://mainnet.base.org/v3
WALLET_ADDRESS=
WALLET_SECRET=
FACTORY_ADDRESS=
SWAP_ROUTER_ADDRESS=
WETH_ADDRESS=
USDC_ADDRESS=
CHAIN_ID=
```

Install `node_modules`
```bash
npm i
```

Run
```
npm run build && npm run start
```
